## 它能做什么

把一部真正的智能手机搬上屏幕：一台停靠、常驻显示的设备，配有主屏幕、状态栏、应用图标、通知、铃声以及可换主题的壁纸。其核心是一个面向 Aetherphone 用户的小型社交网络：Chirper 用于微博、Aethergram 用于分享照片、Phone 用于群组语音通话，全部同属一个社交网络；此外还有一款 Messages 应用，把游戏的 `/tell` 系统变成了一个真正的聊天客户端。

## 功能

### 设备

- **主屏幕与外壳**：一台停靠设备，带有状态栏和多页应用网格。长按进入编辑模式，然后拖动图标，跨页面重新排列。每个屏幕之间都有平滑的滑动过渡。
- **锁屏**：一个真正的锁屏，配有大号时钟与日期、以卡片形式堆叠的最新通知（点按其中一张即可直接跳转到对应应用），以及滑动即可解锁返回。
- **控制中心**：向下滑动即可使用快捷开关：勿扰、位置锁定、待机滚动，另外还有强调色色板、亮度（文字大小）与音量滑块，以及实时音乐控制。
- **通知**：一个通知中心、可选的横幅提示、自定义提醒声音，以及一个勿扰开关。

### 社交

Aetherphone 运行着自己的社交网络 Aethernet，因此这些应用可以跨角色、跨会话工作，而不仅限于本地。

- **Chirper**：一个 X/Twitter 风格的微博。发布简短动态，浏览「为你推荐」和「关注」信息流，搜索、编辑个人资料并举报内容。
- **Aethergram**：一个 Instagram 风格的照片信息流，构建于你的 Photos 图库之上。发布你的截图，关注其他玩家，浏览他们分享的内容。
- **Messages**：读取收到的 `/tell`，并将其排布为可回复的聊天气泡，配有横幅通知以及服务器信息栏上的未读徽章。
- **Phone**：直接从设备向其他 Aetherphone 用户发起群组语音通话。
- **Find People**：在 Lodestone 上查找玩家，并直接与其开始对话。
- **Velvet**：一款独立、私密的 18+ 伴侣应用，用于分享作品并与其他创作者建立联系。以可选构建的形式提供。

### 应用

- **Contacts**：把好友列表当作通讯录，配有 Lodestone 头像；可直接从某个联系人开始对话。
- **Character**：本地角色的资料卡，包含装备、Lodestone 立绘，以及一个健身风格的「活动」仪表盘，追踪职业精通环。
- **Skywatcher**：你所在区域的实时艾欧泽亚天气，附带未来几小时的预报。
- **Venues**：浏览社区场所与活动、夜店、酒吧、拍照点等，配有进行中与即将开始的筛选、标签与名称搜索以及收藏。一键即可用 Lifestream 传送前往（或复制指令），或打开该场所的详情页与 Discord。
- **News**：你所在地区的 Lodestone 信息流：主题、公告、维护与更新，配有图片、维护时段，以及点按打开完整报道。
- **Market**：来自 Universalis 的实时市场板价格。搜索任意物品（或在游戏中右键点击），查看最便宜的挂单、价格统计、销售速度以及近期成交历史，并附有跨你的服务器、大区或地区的趋势图。设置会让手机响铃的降价提醒，与 NPC 商人价格作对比，并收藏喜欢的物品。
- **Wallet**：一目了然地追踪你的金币、货币、点数、狩猎印记与 PvP 战绩点，附带每周上限的进度。
- **Music**：一个游戏内播放器。按类型浏览电台或搜索任意曲目，配有播放控制以及主屏幕上的「正在播放」横幅。
- **Camera**：直接从手机拍摄游戏内画面，支持方形与照片模式、可选的构图网格，以及可切换的闪光灯。
- **Photos**：一个存放你截图的图库，像真正的照片库那样排布，并配有全屏查看器。
- **Games**：一座掌上街机，收录 16 款小游戏，涵盖逻辑、记忆、消消乐、街机、益智、卡牌与策略：Sweeper、Nonogram、Pairs、Simon、Gem Swap、Tetris、2048、Water Sort、Flow、Solitaire、Reversi、Breakout、Bubble Shooter、Snake、Flap 与 Whack，每款都会记录你的最高分。
- **Clock**：一目了然地查看本地、艾欧泽亚与服务器时间，为你添加的任意城市显示世界时钟，另有闹钟、秒表与倒计时。
- **Calendar**：以月视图查看进行中与即将开始的游戏内活动，叠加显示你自己的自定义活动，并可逐日点按查看当天日程。
- **Notes**：快速笔记与可勾选的提醒清单，可选设置到期日，到期时会让手机响铃。
- **Calculator**：一个标准计算器，带有滚动的历史纸带，点按即可复用过往结果。
- **Timers**：距离每日、部队与每周重置的倒计时，另有时尚品鉴、金碟游乐场巨型仙人刺以及大海捕鱼的时段与你的雇员探险，可选提醒。
- **Fishing**：大海捕鱼航程预测，让你在启航前就知道会遇到什么。
- **Dailies**：追踪你循环进行的每日与每周内容，配有可勾选的条目、徽章计数以及自动检测的完成状态。
- **Collections**：浏览你的坐骑、宠物与其他收藏品。
- **Inventory**：无需打开游戏菜单即可快速查看你的背包。
- **Maps**：世界地图与兴趣点。
- **Feedback**：从手机内直接给开发者发送留言。

### 个性化

- **主题与壁纸**：挑选一套强调色配色与一张壁纸：内置美术素材、你自己的某张照片，或任意导入的图片。整台设备都会随之改变外观以相匹配。
- **Lodestone 立绘**：真实的角色头像与立绘，从 Lodestone 拉取，显示在你的个人资料、联系人与聊天中（可开关）。
- **文字大小**：一个无障碍缩放选项，在不改变手机尺寸的前提下缩放设备上的文字。
- **声音**：为通话设置铃声、为通知设置提醒音，可从内置库或你自己导入的 MP3/WAV 文件中选择。
- **待机动画**：当你站着不动时，你的角色会悠闲地滑动手机（Tomescroll），可选。
- **关于窗口**：一个带动画的鸣谢与链接页面，可从设置或通过 `/phone about` 打开。

## 安装

在游戏中：`/xlsettings` → **Experimental** → 粘贴到 **Custom Plugin Repositories**：

```
https://raw.githubusercontent.com/XeldarAlz/DalamudPlugins/main/repo.json
```

勾选 **Enabled**，点击 **+**，然后点击 **Save and Close**。打开 `/xlplugins` → **All Plugins**，搜索 **Aetherphone** 并安装。

## 指令

| 指令 | 作用 |
|---|---|
| `/phone` | 切换手机显示 |
| `/aetherphone` | `/phone` 的别名 |
| `/phone about` | 打开鸣谢 / 链接 |
| `/phone reset` | 将手机重新居中于屏幕 |

## これは何か

Aetherphone は、FINAL FANTASY XIV の画面上に本物のスマートフォンを置く、無料でオープンソースの Dalamud プラグインです。ホーム画面、通知、着信音、テーマ変更できる壁紙を備えた、ドッキングされ常時表示されるデバイスです。アプリの背後では Aetherphone ユーザー向けの独自のソーシャルネットワークが動作しており、ローカルだけでなくキャラクターやセッションをまたいで機能します。

プライバシーと安全性を最優先しています。テキストメッセージ・添付ファイル・ボイスメモはエンドツーエンドで暗号化され、通話は通信経路上で暗号化されます。投稿と画像は明確なコンテンツルールのもとで AI モデレーションを通ります。

## ハイライト

- **ソーシャル**：Chirper のマイクロブログ、Aethergram の写真フィード、ボイスメモとグループ通話に対応したプライベートメッセージの ChocoChat、そして任意の 18+ コンパニオンアプリ Velvet。
- **ツール**：マーケットボードのトラッカー、会場・イベントの一覧、ゲーム内ミュージック、天気、ウォレット、タイマーとリセット、フォトライブラリとカメラ、そしてミニゲームのポケットアーケードなど、30 以上のアプリ。
- **自分好みに**：アクセントパレット、壁紙、Lodestone のキャラクターポートレート、カスタム着信音、そして文字サイズのアクセシビリティズーム。

機能の全ツアー、スクリーンショット、詳細はウェブサイトでご覧いただけます。

→ **[www.aetherphone.net](https://www.aetherphone.net/)**

## インストール

ゲーム内で：`/xlsettings` → **Experimental** → **Custom Plugin Repositories** に貼り付け：

```
https://raw.githubusercontent.com/XeldarAlz/DalamudPlugins/main/repo.json
```

**Enabled** にチェックを入れ、**+** をクリックし、**Save and Close** を押します。`/xlplugins` → **All Plugins** を開き、**Aetherphone** を検索してインストールします。

## コマンド

| コマンド | 動作 |
|---|---|
| `/phone` | 電話の表示を切り替える |
| `/aetherphone` | `/phone` のエイリアス |
| `/phone about` | クレジット / リンクを開く |
| `/phone reset` | 電話を画面中央に戻す |

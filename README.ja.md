## 何ができるか

本物のスマートフォンを画面上に置きます。ドッキングされ常時表示されるデバイスで、ホーム画面、ステータスバー、アプリアイコン、通知、着信音、テーマ変更できる壁紙を備えています。その中核にあるのは Aetherphone ユーザー向けの小さなソーシャルネットワークです。マイクロブログの Chirper、写真を共有する Aethergram、グループ音声通話の Phone がすべて同じソーシャルネットワークの一部として動作し、さらにゲームの `/tell` システムを本格的なチャットクライアントに変える Messages アプリも並んでいます。

## 機能

### デバイス

- **ホーム画面とシェル**：ステータスバーと複数ページのアプリグリッドを備えた、ドッキングされたデバイス。長押しで編集モードに入り、アイコンをドラッグしてページをまたいで並べ替えられます。すべての画面間で滑らかなスライド遷移。
- **ロック画面**：大きな時計と日付、カードとして積み重なった最新の通知（タップすればそのアプリへ直接ジャンプ）、そしてスワイプで戻れる、本物のロック画面。
- **コントロールセンター**：下にスワイプするとクイックトグルが使えます。おやすみモード、位置ロック、アイドルスクロール、加えてアクセントカラーのスウォッチ、明るさ（文字サイズ）と音量のスライダー、そしてライブな音楽コントロール。
- **通知**：通知センター、オプションのトースト、カスタムのアラート音、そしておやすみモードのスイッチ。

### ソーシャル

Aetherphone は独自のソーシャルネットワーク Aethernet を運営しているため、これらのアプリはローカルだけでなく、キャラクターやセッションをまたいで機能します。

- **Chirper**：X/Twitter 風のマイクロブログ。短い近況を投稿し、「あなたへのおすすめ」と「フォロー中」のフィードを閲覧し、検索、プロフィール編集、コンテンツの報告ができます。
- **Aethergram**：あなたの Photos ライブラリを基にした Instagram 風の写真フィード。撮影した写真を投稿し、他のプレイヤーをフォローし、彼らが共有したものを閲覧できます。
- **Messages**：受信した `/tell` を読み取り、返信できるチャットの吹き出しとして並べます。トースト通知と、サーバー情報バー上の未読バッジ付き。
- **Phone**：デバイスから直接、他の Aetherphone ユーザーへグループ音声通話を発信できます。
- **Find People**：Lodestone でプレイヤーを検索し、そのまま会話へ移れます。
- **Velvet**：作品を共有し、他のクリエイターとつながるための、独立した非公開の 18+ コンパニオンアプリ。オプションのビルドとして提供されます。

### アプリ

- **Contacts**：フレンドリストを住所録として扱い、Lodestone のポートレート付きで表示します。連絡先から直接会話を始められます。
- **Character**：ローカルキャラクターのプロフィールカード。装備、Lodestone のポートレート、そしてジョブの熟練リングを追跡するフィットネス風のアクティビティダッシュボード付き。
- **Skywatcher**：現在のゾーンのリアルタイムなエオルゼア天候。これから数時間の予報付き。
- **Venues**：コミュニティの会場やイベント、ナイトクラブ、バー、撮影スポットなどを閲覧できます。開催中・開催予定のフィルター、タグと名前での検索、お気に入り付き。ワンタップで Lifestream を使ってそこへ移動（またはコマンドをコピー）でき、会場の掲載ページと Discord を開くこともできます。
- **News**：あなたの地域の Lodestone フィード。トピックス、お知らせ、メンテナンス、アップデートを、画像やメンテナンス時間帯とともに表示し、タップで記事全文を開けます。
- **Market**：Universalis からのリアルタイムなマーケットボード価格。任意のアイテムを検索（またはゲーム内で右クリック）し、最安の出品、価格統計、売れ行き、直近の販売履歴を、ワールド・データセンター・地域ごとのトレンドグラフとともに確認できます。電話を鳴らす値下がりアラートを設定し、NPC ベンダー価格と比較し、お気に入りをスターできます。
- **Wallet**：ギル、各種通貨、トークン、狩猟章、PvP 戦績を一目で把握でき、週間上限までの進捗も確認できます。
- **Music**：ゲーム内プレイヤー。ジャンル別ステーションを閲覧したり任意の曲を検索したりでき、再生コントロールとホーム画面の「再生中」バナー付き。
- **Camera**：ゲーム内のショットを電話から直接撮影できます。スクエアと写真のモード、オプションの構図グリッド、切り替え可能なフラッシュ付き。
- **Photos**：撮影した写真のギャラリー。本物のフォトライブラリのように並び、全画面ビューアー付き。
- **Games**：ロジック、記憶、マッチ3、アーケード、パズル、カード、ストラテジーにわたる 16 種のミニゲームを収めたポケットアーケード。Sweeper、Nonogram、Pairs、Simon、Gem Swap、Tetris、2048、Water Sort、Flow、Solitaire、Reversi、Breakout、Bubble Shooter、Snake、Flap、Whack。それぞれあなたのベストスコアを記録します。
- **Clock**：ローカル、エオルゼア、サーバーの時刻を一目で確認でき、追加した任意の都市のワールドクロック、さらにアラーム、ストップウォッチ、カウントダウンタイマー付き。
- **Calendar**：開催中・開催予定のゲーム内イベントを月表示で確認でき、自分のカスタムイベントを重ねて表示し、各日のアジェンダをタップで辿れます。
- **Notes**：クイックメモとチェックできるリマインダーリスト。期日を任意で設定でき、時が来ると電話を鳴らします。
- **Calculator**：標準的な電卓。流れる履歴テープ付きで、タップすれば過去の結果を再利用できます。
- **Timers**：デイリー、グランドカンパニー、ウィークリーのリセットまでのカウントダウン、さらにファッションチェック、ジャンボクジ、オーシャンフィッシングの時間帯やリテイナーの探索も、任意のリマインダー付きで表示します。
- **Fishing**：オーシャンフィッシングの航海予測。出航前に何が来るかが分かります。
- **Dailies**：繰り返し行うデイリー・ウィークリーのコンテンツを追跡します。チェックできる行、バッジのカウント、自動検出される達成状況付き。
- **Collections**：マウント、ミニオン、その他の収集品を閲覧できます。
- **Inventory**：ゲームメニューを開かずに、バッグの中身をさっと確認できます。
- **Maps**：ワールド内のマップと見どころ。
- **Feedback**：電話の中から開発者へ直接メッセージを送れます。

### パーソナライズ

- **テーマと壁紙**：アクセントパレットと壁紙を選べます。内蔵のアート、自分の写真の一枚、または取り込んだ任意の画像。デバイス全体がそれに合わせてスタイルを変えます。
- **Lodestone ポートレート**：本物のキャラクターアバターとポートレートを Lodestone から取得し、プロフィール、連絡先、チャットに表示します（切り替え可能）。
- **文字サイズ**：電話のサイズは変えずに、デバイス上の文字を拡大縮小するアクセシビリティのズーム。
- **サウンド**：通話用の着信音と通知用のアラート音を設定できます。内蔵ライブラリ、または取り込んだ自分の MP3/WAV ファイルから選べます。
- **アイドルアニメーション**：立ち止まっているとき、キャラクターがのんびり電話をスクロールします（Tomescroll）。任意。
- **Aboutウィンドウ**：アニメーション付きのクレジットとリンクの画面。設定から、または `/phone about` で開けます。

## インストール

ゲーム内で：`/xlsettings` → **Experimental** → **Custom Plugin Repositories** に貼り付け：

```
https://raw.githubusercontent.com/XeldarAlz/DalamudPlugins/main/repo.json
```

**Enabled** にチェックを入れ、**+** をクリックし、**Save and Close** を押します。`/xlplugins` → **All Plugins** を開き、**Aetherphone** を検索してインストールします。

## コマンド

| コマンド | 動作 |
|---|---|
| `/phone` | 電話の表示を切り替える |
| `/aetherphone` | `/phone` のエイリアス |
| `/phone about` | クレジット / リンクを開く |
| `/phone reset` | 電話を画面中央に戻す |
